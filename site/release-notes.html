Release Notes - Portals Bridges - Version 1.0.4

The list below outlines the issues that are addressed with release 1.0.4.

The release notes of the previous release 1.0.3 are available here: release notes 1.0.3.

Bug

  • [PB-71] - FacesPortlet should apply RenderKitFactory.HTML_BASIC_RENDER_KIT when default-render-kit-id is not specified
  • [PB-72] - ClassCastException in PortletApplicationModelImpl since Spring Framework 2.0.5

Improvement

  • [PB-74] - Safe creation of context in GenericVelocityPortlet for JS2's new feature, auto-dispatching for edit_defaults mode.
  • [PB-75] - GroovyPortlet need to support Jetspeed's new feature, auto-dispatching for edit_defaults mode option.
